# متجر بصمة ثلاثية الأبعاد 🧊

موقع بسيط (HTML/CSS/JavaScript بدون سيرفر) لعرض المجسمات ثلاثية الأبعاد مع:
- عرض المنتجات: الصورة، الاسم، الوصف، السعر، **وكود المنتج**.
- سلة لاختيار عدة منتجات وحساب الإجمالي.
- زر **إرسال الطلب عبر واتساب** يفتح محادثة البائع برسالة جاهزة فيها المنتجات (مع أكوادها) والإجمالي.
- لوحة إدارة (عبر رابط خاص) لإضافة/تعديل/حذف المنتجات، **ورفع الصور من الجهاز**، **وتعديل اسم الموقع وشعاره**.

## الرابطان (مهم)
- **رابط الزبائن (عام):** `https://اسم-حسابك.github.io/اسم-المستودع/`
  الزائر العادي لا يرى زر الإدارة إطلاقًا.
- **رابط الإدارة (خاص بك):** أضف `#admin` في نهاية الرابط:
  `https://اسم-حسابك.github.io/اسم-المستودع/#admin`
  عندها يظهر زر **الإدارة ⚙️** (ويبقى محميًا بكلمة مرور أيضًا).

## الملفات
| الملف | الوظيفة |
|------|---------|
| `index.html` | الصفحة الرئيسية |
| `styles.css` | التنسيق |
| `app.js` | المنطق والإعدادات |
| `products.json` | قائمة المنتجات (مصدر البيانات للجميع) |
| `settings.json` | اسم الموقع والشعار (للجميع) |

## الإعدادات السريعة
أعلى ملف `app.js` داخل `CONFIG`:
- `whatsappNumber`: رقم واتساب البائع بالصيغة الدولية بدون `+` أو `00`. الحالي: `96877280090`.
- `currency`: العملة (الافتراضي `ر.ع` — ريال عماني).
- `adminPassword`: كلمة مرور دخول وضع الإدارة (الافتراضي `admin` — **يُفضّل تغييرها**).

## كيف تُدير المنتجات
1. افتح **رابط الإدارة** (`...#admin`)، اضغط زر **الإدارة ⚙️** وأدخل كلمة المرور.
2. **إضافة/تعديل منتج**: الاسم، الوصف، السعر، **كود المنتج**، و**صور تُرفع من جهازك** (يمكن اختيار **أكثر من صورة** للمنتج الواحد وتظهر كمعرض بأسهم ونقاط على البطاقة). كل صورة تُصغّر تلقائيًا، ولكل صورة زر إزالة. يمكن أيضًا إضافة صورة برابط أو استخدام أيقونة من "خيارات متقدمة".
3. اضغط **⬇️ تصدير المنتجات** لتنزيل `products.json` المحدّث، ثم ارفعه على GitHub (استبدل القديم) لتظهر التعديلات لكل الزوار.
4. زر **↺ استرجاع** يلغي تعديلاتك المحلية ويعيد التحميل من `products.json`.

## تعديل اسم الموقع والشعار
1. من وضع الإدارة اضغط **⚙️ إعدادات الموقع**.
2. عدّل **اسم الموقع** و**الوصف**، وارفع **صورة الشعار** من جهازك (أو استخدم إيموجي).
3. اضغط **حفظ** (يظهر فورًا)، ثم **⬇️ تصدير** لتنزيل `settings.json` وارفعه على GitHub لتثبيت الاسم والشعار لكل الزوار.

> ملاحظة: لأن الموقع ثابت (Static)، تعديلاتك داخل المتصفح تخصّك أنت فقط حتى ترفع ملفات `products.json` / `settings.json` المحدّثة على GitHub. الصور المرفوعة تُحفظ داخل هذه الملفات (يكبر حجمها قليلًا حسب عدد الصور).

## النشر على GitHub Pages (مجاني)
1. أنشئ مستودعًا جديدًا على GitHub وارفع كل الملفات (`index.html`, `styles.css`, `app.js`, `products.json`, `settings.json`).
2. من المستودع: **Settings → Pages**.
3. في **Source** اختر الفرع `main` والمجلد `/ (root)` ثم **Save**.
4. انتظر دقيقة، وسيظهر رابط موقعك العام:
   `https://اسم-حسابك.github.io/اسم-المستودع/`
   ورابط الإدارة: نفس الرابط + `#admin`.

## تجربة الموقع محليًا
- افتح `index.html` مباشرةً في المتصفح (سيستخدم القيم الاحتياطية إذا مُنع تحميل ملفات JSON).
- أو شغّل خادمًا محليًا لقراءة `products.json` و`settings.json` بشكل كامل:
  ```bash
  python -m http.server 8000
  ```
  ثم افتح `http://localhost:8000` (والإدارة على `http://localhost:8000/#admin`).
